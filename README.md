# GitHub Upload Tools

مجموعة من السكريبتات الذكية لرفع المشاريع إلى GitHub بسهولة وأمان.

## 🚀 الميزات

- ✅ رفع ذكي للمشاريع إلى GitHub
- ✅ معالجة تلقائية للملفات المهمة (Makefile, Dockerfile, إلخ)
- ✅ إنشاء ملفات README و LICENSE تلقائياً
- ✅ دعم GitHub CLI والطريقة التقليدية
- ✅ عرض تفصيلي للملفات المرفوعة والمتجاهلة
- ✅ إنشاء ملف .gitignore تلقائياً إذا لم يكن موجوداً

## 📦 السكريبتات المتاحة

### 1. `direct-upload.sh` (الأفضل للرفع المباشر) 🎯
السكريبت المخصص لرفع الملفات مباشرة في جذر المستودع:

```bash
./direct-upload.sh
```

**الميزات:**
- رفع الملفات مباشرة في جذر المستودع
- تحقق من المجلد الصحيح
- معالجة ذكية للملفات المهمة
- إنشاء ملف .gitignore محسن
- دعم GitHub CLI
- عرض تفصيلي للملفات

### 2. `extract-and-upload.sh` (للملفات المضغوطة) 📦
السكريبت المخصص لاستخراج الملفات المضغوطة ورفعها مباشرة:

```bash
./extract-and-upload.sh
```

**الميزات:**
- استخراج الملفات المضغوطة (ZIP)
- رفع محتويات المشروع مباشرة في جذر المستودع
- معالجة المجلدات الفرعية تلقائياً
- تنظيف الملفات المؤقتة
- دعم GitHub CLI

### 3. `smart-upload.sh` (ذكي شامل) 🧠
السكريبت الذكي الأكثر تطوراً:

```bash
./smart-upload.sh
```

**الميزات:**
- معالجة ذكية للملفات المهمة
- إنشاء ملف .gitignore تلقائياً
- دعم GitHub CLI
- عرض تفصيلي للملفات
- معالجة الأخطاء

### 4. `setup-github.sh` (متقدم) ⚙️
السكريبت المتقدم مع ميزات إضافية:

```bash
./setup-github.sh
```

**الميزات:**
- إعداد Git كامل
- إنشاء ملفات CONTRIBUTING.md
- إنشاء package.json إذا لم يكن موجوداً
- دعم GitHub CLI

### 5. `quick-upload.sh` (سريع) ⚡
السكريبت السريع للرفع السريع:

```bash
./quick-upload.sh
```

**الميزات:**
- رفع سريع
- معالجة الملفات المهمة
- عرض الملفات المرفوعة

```bash

```

## 🛠️ حل المشاكل الشائعة

### المشكلة: الملفات تظهر في مجلد فرعي بدلاً من جذر المستودع
**الحل:** استخدم السكريبت المخصص `direct-upload.sh` أو `extract-and-upload.sh` للملفات المضغوطة.

### المشكلة: الملفات لا تظهر في GitHub
**الحل:** استخدم السكريبت الذكي `smart-upload.sh` الذي يتعامل مع هذه المشكلة تلقائياً.

### المشكلة: خطأ في المصادقة
**الحل:** تأكد من تسجيل الدخول إلى GitHub CLI:
```bash
gh auth login
```

### المشكلة: ملف Makefile لا يظهر
**الحل:** السكريبتات الجديدة تضيف ملف Makefile تلقائياً حتى لو كان في .gitignore.

### المشكلة: ملفات المشروع في مجلد مضغوط
**الحل:** استخدم `extract-and-upload.sh` لاستخراج الملفات ورفعها مباشرة في جذر المستودع.
